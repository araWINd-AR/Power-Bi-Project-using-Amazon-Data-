# Power BI Project using Amazon Data

This repository contains a Power BI project built using Amazon-related data. The main goal of this project is to analyze and visualize the dataset in a meaningful way through an interactive Power BI dashboard.

I created this project to practice data cleaning, data modeling, dashboard design, and visual storytelling in Power BI using a real dataset. It reflects my hands-on learning process in business intelligence and data analytics.

## Project Overview

This project is based on an Amazon dataset stored in Excel and developed into a Power BI report. The dashboard is intended to transform raw data into a more understandable and visual format so that trends, patterns, and insights can be explored more easily.

The project also includes a logo image used for dashboard presentation and branding.

## Files in This Repository

This repository currently includes the following files:

- **Amazon_power_bi_project.pbix** – the main Power BI dashboard/report file
- **amazon_prime_titles.xlsx** – the source dataset used for analysis
- **Prime video logo.png** – image used in the report for visual presentation

## Objective of the Project

The main objective of this project is to:

- work with a real dataset in Power BI
- create a structured and interactive dashboard
- practice transforming raw data into useful insights
- improve skills in reporting, visualization, and dashboard formatting
- strengthen hands-on knowledge in Power BI and data analytics

## What This Project Demonstrates

Through this project, I worked on:

- importing and connecting external data files
- preparing the dataset for reporting
- building visuals in Power BI
- organizing the report in a cleaner and more readable format
- improving the dashboard presentation using formatting and design elements
- using visual analysis to better understand the dataset

## Tools Used

- **Power BI Desktop**
- **Microsoft Excel**
- **Power Query**
- **DAX** (where applicable)

## About the Dataset

The dataset used in this project is saved as **amazon_prime_titles.xlsx**. It is used as the source file for the Power BI report. The report is built to analyze the data visually and make it easier to interpret through charts, filters, and dashboard elements.

I built this project as part of my Power BI practice and learning journey. Instead of only learning concepts theoretically, I wanted to apply them through a project that involves an actual dataset and a complete report file.

This project helped me improve my understanding of:

- how to work with data inside Power BI
- how to create a report from raw data
- how to present analysis in a more professional and visual way
- how to organize project files for sharing on GitHub

## How to Use This Repository

1. Download or clone this repository
2. Open the **Amazon_power_bi_project.pbix** file in Power BI Desktop
3. Make sure the dataset file **amazon_prime_titles.xlsx** is available in the correct path if Power BI asks for the source
4. Explore the report and visuals inside Power BI
